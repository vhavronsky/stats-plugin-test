# Welcome to Local AI Worker Docs

The Local AI Worker provides a robust interface for managing and monitoring AI task execution in local environments. This documentation helps you integrate worker health monitoring and user management into your applications.

## Key Components

The worker system consists of two primary modules:
- **Core Worker Service**: Handles health checks and system status.
- **Identity Management**: Manages user access and profiles.
